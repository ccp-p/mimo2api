快速开始 — 如何获取 Cookie

1
打开 https://aistudio.xiaomimimo.com，登录你的小米账号
2
按 F12 打开开发123者工具，切换到 Network 标签
3
在聊天框发送任意消息，筛选 chat 请求
4
右键该请求 → Copy as cURL，粘贴到下方解析框
5
点击「解析 cURL」自动提取凭证，然后添加账号即可

在 Claude Code 中使用

# 设置环境变量（也可写入 .bashrc / .zshrc）
export ANTHROPIC_BASE_URL="http://localhost:8080/v1"
export ANTHROPIC_API_KEY="sk-mimo" # 可在启动时通过 -key 参数自定义

# 启动 Claude Code
claude --model mimo-v2-flash-studio

支持的模型名：mimo-v2-flash-studio（默认） · mimo-v2-flash-studio-thinking（深度思考） · mimo-v2-flash-studio-search（联网搜索） · mimo-v2-flash-studio-thinking-search（思考+搜索）
